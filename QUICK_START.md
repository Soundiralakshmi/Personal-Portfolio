# Quick Start - 5 Minute Setup

## ⚡ Fastest Path to Live Portfolio

### 1. Get a Free Database (2 minutes)
Go to **https://planetscale.com** and:
- Sign up (free)
- Create database "portfolio_db"
- Copy your credentials (host, user, password)

### 2. Add to Vercel (2 minutes)
1. Visit: https://vercel.com/soundiras-projects/mypersonal-port/settings/environment-variables
2. Click "Add New"
3. Paste these (replace with YOUR credentials):

| Key | Value |
|-----|-------|
| DB_HOST | your-host.mysql.planetscale.com |
| DB_USER | your-username |
| DB_PASSWORD | your-password |
| DB_NAME | portfolio_db |
| DB_PORT | 3306 |
| NODE_ENV | production |

4. Click "Save"
5. Go to **Deployments** → Click latest → **Redeploy**

### 3. Done! ✅
Your portfolio is now LIVE at: **https://mypersonal-port.vercel.app**
